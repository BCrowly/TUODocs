## For Mac OS, courtesy of Pretzl   

Clone TazUO:

<Code code='git config --global url."https://".insteadOf git://' lang="bash" />

<Code code="git clone https://github.com/bittiez/TazUO.git --recursive" lang="bash" />

<Code code="cd TazUO/" lang="bash" />

<Code code="nuget restore" lang="bash" />

<Code code="dotnet publish -c Release" lang="bash" />

Navigate to `TazUO/bin/Release`

<Code code="chmod +x ClassicUO" lang="bash" />

<Code code="./ClassicUO" lang="bash" />
