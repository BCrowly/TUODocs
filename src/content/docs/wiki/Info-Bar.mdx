## Graphic option
![image](https://github.com/bittiez/ClassicUO/assets/3859393/1a56ce4a-9031-4340-a0f8-289b8e918da1)  
To use this feature, you need to change the label to `\` followed by the graphic number you want to use (Item graphics only, not gump)  

![image](https://github.com/bittiez/ClassicUO/assets/3859393/7bf1a943-03f0-4bde-90f6-ff11d7294c1b)  

No other text can be present, and you need to use the integer graphic number, not the 0x0345 format like the screenshot above.

## Font settings
![image](https://github.com/bittiez/TazUO/assets/3859393/c8b98a75-9a4e-4971-b3e1-240a97d539b0)  
Font settings for InfoBar are located under `Options->TazUO->Misc`

## Resizable
![Animation](https://github.com/bittiez/TazUO/assets/3859393/e267e763-30b8-4255-9ad9-1f95bec1161e)  
You can now resize the infobar as desired, then hold alt and click the lock in the top-left corner to hide the border and lock it in place.
