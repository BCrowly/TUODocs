## Features
* Easier to read
* Resizable
* Has customizable(v3.11+) tabs for different message types
* Customize background color and opacity
* Can be anchored to world map or grid containers
* Can adjust max journal entries from 200-2000 as desired.
* Supports global chat used on OSI and some private servers

## Settings
You can adjust the journal opacity, background color, and choice of gump style under `Options->TazUO->Journal`
