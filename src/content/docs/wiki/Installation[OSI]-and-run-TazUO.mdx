## How to Install and Configure UOStealth

UOStealth is a third-party tool for Ultima Online that allows players to automate tasks, create scripts, and enhance gameplay. Follow the steps below to install and configure UOStealth properly.


### Download UOStealth

Visit the [official UOStealth website](https://stealth.od.ua/).
Download the latest version of the UOStealth installer.


#### Install and Configure UOStealth
After downloading the ZIP file, extract its contents to your desired folder.

![image](https://github.com/user-attachments/assets/93af2a44-2e7a-406f-8dc8-9bd15a3a398a)


![image](https://github.com/user-attachments/assets/15222acd-0879-49bd-b733-c41455ede34b)

Run the Stealth.exe file.

![image](https://github.com/user-attachments/assets/d708dfd4-9520-48a5-a2ae-06eb2d077be1)

You'll need to configure your original Ultima Online client and profiles. To do this, follow these steps:


![image](https://github.com/user-attachments/assets/d1bd2406-7608-4b37-921a-df46cd893bdc)

Click on the Profile Configuration icon.

![image](https://github.com/user-attachments/assets/c1b1d1e5-dfc7-4ac7-86be-f4cbc39c70ea)

Next, click the Shard Information Configuration icon.

![image](https://github.com/user-attachments/assets/f3b49a86-704d-42f1-b549-dfee3c4e5b67)

Now, click the Tazuo Version Configuration icon.

![image](https://github.com/user-attachments/assets/4a37571b-69fe-4ece-8c37-434fbb28f83b)

You will need to specify the installation folder for your Tazuo client:

Enter a name for your configuration and the path to your Tazuo folder.

Click the + button and then click OK to save your configuration.

_**Don't lost active encryption client**_

![image](https://github.com/user-attachments/assets/50e0876f-c969-4ee7-8853-7a54cd53ecd6)

Next, configure access to the server:
![image](https://github.com/user-attachments/assets/12b41fa5-e4e3-4b6c-8371-1b4ef32c60a8)

In the Server Name, Address, and Port, enter: login.ultimaonline.com port 7776
For the Client Version, run your original Ultima Online client to find the required version and input it here.
In the MUL or UOP File Path, point to your original client’s directory.
In the Select Client dropdown, choose the client you just configured.

Once everything is set up, click Add and then Save.

![image](https://github.com/user-attachments/assets/df2432d1-5a8b-4c9d-bba6-74e3bd70d590)

This is all configure ok

![image](https://github.com/user-attachments/assets/496c3665-0229-4fdf-8084-ce41a3742265)

####  Connecting to the Server
Once your configuration is complete, click Connect.

![image](https://github.com/user-attachments/assets/b3b17c90-c700-44ec-8199-0a5babc4ead7)

For connect in your tazuo client just need click in start client.
