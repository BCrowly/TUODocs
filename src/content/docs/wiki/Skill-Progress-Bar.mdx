You can enable skill progress bars to more easily see skill changes

## Options  
Navigate to `Options -> TazUO -> Misc` and you will see the following options:  
![image](https://github.com/bittiez/TazUO/assets/3859393/b4d84f32-3638-4cee-9efb-19fbd0a9212d)

The format option allows you to use all or none of these:  
`{0}` = Skill name  
`{1}` = Skill value  
`{2}` = Skill cap
